### Lv2 진화 (개별 분기)

각 Lv1 타워는 고유한 두 경로 중 하나로 진화합니다.
진화 비용 = 기본 비용 × (기본타워cost / 50).

| 타워 | 분기 A | 분기 B |
|------|--------|--------|
| **태양** | 준거성 (80) — 사거리+1, 데미지+30% | 백색왜성 (100) — 사거리=2, 데미지×3 |
| **프록시마** | 플레어 성 (70) — 속사 유지 + 5초마다 범위 버스트 (dmg×2) | 연성계 (60) — 2연발 (각 70% 데미지) |
| **시리우스** | 시리우스 B (100) — 장갑 관통+5, 데미지+40% | A형 초거성 (120) — 사거리+1.5, 반경 2.5 버프 오라 (+15% 데미지) |
| **리겔** | 청색 초거성 (130) — 데미지×1.8, 사거리+1, 공속×0.6 | 적색 초거성 (100) — 데미지 유지, 스플래시 반경 1.0 추가 |
| **베텔게우스** | 초신성 전조 (80) — 3웨이브 폭발, 데미지+50%, 사거리+0.5 | 맥동 변광성 (90) — 폭발 포기 → 3초 주기 범위 펄스 (30dmg, 반경2) |
| **볼프-레이에** | WC형 (100) — 스플래시↑ 2.0, 공속×0.7 | WN형 (110) — 직격×1.5, 공속×1.4, 스플래시↓ 1.0 |
| **마그네타** | SGR 반복체 (120) — 통상 사격 + 10초마다 관통 감마선 (dmg×3) | 밀리초 펄서 (100) — 공속×3, 데미지×0.4 |
