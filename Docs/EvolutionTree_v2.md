# 진화 트리 v2 — 개별 분기 설계

## 설계 원칙
각 별은 자신의 질량과 분광형에 따라 고유한 운명을 걷는다.
M형 왜성이 청색거성이 되는 것은 불가능 — 천문학적 고증 기반 개별 경로.

## Lv1 → Lv2 (고유 분기)

### Sol (G2V, 균형형)
- **준거성** (Subgiant, G-IV) — 공속 유지, 사거리+1, 데미지 소폭↑. 태양형 팽창 초기
- **백색왜성** (White Dwarf, DA) — 사거리↓ 2타일, 데미지×3, 관통. 초고밀도

### Proxima (M5V, 속사형)
- **플레어 성** (Flare Star) — 속사 유지 + 5초마다 범위 미니 플레어 버스트
- **연성계** (Binary System) — 2연발 사격 (듀얼샷, 각 70% 데미지)

### Sirius (A1V, 중거리 딜러)
- **시리우스 B** (White Dwarf Companion) — 방어 관통+5, 공격력↑. 백색왜성 동반성
- **A형 초거성** (A-Supergiant) — 사거리↑, 공속↓, 주변 타워 공격력 +15% 버프 오라

### Rigel (B8Ia, 스나이퍼)
- **청색 초거성** (Blue Supergiant) — 데미지×1.8, 사거리+1, 공속↓↓. 극한 원샷
- **적색 초거성** (Red Supergiant) — 데미지 유지, 스플래시 반경 1.0 추가

### Betelgeuse (M1Ia, 초신성 순환)
자폭 ≠ 손실. 폭발(II형 초신성)은 광역 데미지 후 그 자리에 **초신성 잔해(영구 DoT 존)**를 남기고, 타일은 재건축 가능 — 별의 순환을 플레이로 옮긴 설계.
- 기본: 5웨이브 후 폭발 — 반경 3타일, 공격력×12.5 (기본 100) 데미지, 잔해 3dps·반경 1.5 잔류
- **초신성 전조** (Pre-Supernova) — 폭발까지 3웨이브, 반경 4타일, 150 데미지, 강화 잔해 (5dps, 반경 2.5)
- **맥동 변광성** (Pulsating Variable) — 폭발 무기한 연기 (현재의 베텔게우스 그 자체 — 반규칙 변광성), 3초 주기 범위 펄스 (30dmg, 반경 2)

### Wolf-Rayet (WN8, 스플래시)
- **WC형** (Carbon WR) — 스플래시 반경↑ 2.0, 범위 피해 80%, 공속↓
- **WN형** (Nitrogen WR) — 스플래시 반경↓ 1.0, 직격×1.5, 공속↑

### Magnetar (SGR, 고급)
- **소프트 감마선 반복체** (SGR Repeater) — 10초마다 직선 관통 감마선 버스트
- **밀리초 펄서** (Millisecond Pulsar) — 공속×3, 데미지×0.4

## Lv2 → Lv3 (최종 진화, 일부만)

| Lv2 출발 | Lv3 도달 | 효과 |
|----------|----------|------|
| 준거성 | 행성상 성운 | 방어 감쇠 오라 (태양형 별의 실제 운명) |
| 백색왜성 | Ia형 초신성 | 시한 폭발 (찬드라세카르 한계 초과) |
| 청색 초거성 | 블랙홀 | 근접 즉사 |
| 적색 초거성 | 초신성 잔해 | DoT 영역 |
| 맥동 변광성 | OH/IR 성 | 먼지 껍질 — 방어 감쇠 오라 |
| 밀리초 펄서 | 펄서 | 넉백 + 기절 |
| SGR 반복체 | 블랙홀 | 근접 즉사 |

## 고증 노트 (2026-07 베텔게우스 리워크)
- **행성상 성운 이관**: 행성상 성운은 저·중질량 별(≲8 M☉)의 최후 — M형 초거성 라인(베텔게우스)에서는 물리적으로 불가능. 태양의 실제 운명(준거성→적색거성→행성상 성운)에 맞춰 Sol 준거성의 Lv3로 이동.
- **OH/IR 성**: 맥동하는 M형 (극)초거성이 극단적 질량 방출로 먼지 고치에 싸이는 실제 단계 (VY CMa류). 맥동 변광성의 Lv3로 고증에 부합.
- **초신성 = 순환**: 베텔게우스의 폭발은 타워 소멸이 아니라 초신성 잔해(영구 DoT 존)로의 전환. 폭발 데미지는 공격력×12.5로 스케일 — '오리온의 검' 시너지(폭발+30%)와 열죽음 돌연변이 '초신성 잔광'(피해 2배·3웨이브 단축)이 실제로 적용된다.
