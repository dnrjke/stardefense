# 진화 트리 v2 — 개별 분기 설계

## 설계 원칙
각 별은 자신의 질량과 분광형에 따라 고유한 운명을 걷는다.
M형 왜성이 청색거성이 되는 것은 불가능 — 천문학적 고증 기반 개별 경로.

## Lv1 → Lv2 (고유 분기)

### Sol (G2V, 균형형)
- **준거성** (Subgiant, G-IV) — 공속 유지, 사거리+1, 데미지 소폭↑. 태양형 팽창 초기
- **백색왜성** (White Dwarf, DA) — 사거리↓ 2타일, 데미지×3, 관통. 초고밀도

### Proxima (M5V, 속사형)
- **플레어 성** (Flare Star) — 속사 유지 + 5초마다 범위 미니 플레어 버스트
- **연성계** (Binary System) — 2연발 사격 (듀얼샷, 각 70% 데미지)

### Sirius (A1V, 중거리 딜러)
- **시리우스 B** (White Dwarf Companion) — 방어 관통+5, 공격력↑. 백색왜성 동반성
- **A형 초거성** (A-Supergiant) — 사거리↑, 공속↓, 주변 타워 공격력 +15% 버프 오라

### Rigel (B8Ia, 스나이퍼)
- **청색 초거성** (Blue Supergiant) — 데미지×1.8, 사거리+1, 공속↓↓. 극한 원샷
- **적색 초거성** (Red Supergiant) — 데미지 유지, 스플래시 반경 1.0 추가

### Betelgeuse (M1Ia, 시한폭탄)
- **초신성 전조** (Pre-Supernova) — 폭발까지 3웨이브, 반경 4타일, 150 데미지
- **맥동 변광성** (Pulsating Variable) — 폭발 포기, 3초 주기 범위 펄스 (30dmg, 반경 2)

### Wolf-Rayet (WN8, 스플래시)
- **WC형** (Carbon WR) — 스플래시 반경↑ 2.0, 범위 피해 80%, 공속↓
- **WN형** (Nitrogen WR) — 스플래시 반경↓ 1.0, 직격×1.5, 공속↑

### Magnetar (SGR, 고급)
- **소프트 감마선 반복체** (SGR Repeater) — 10초마다 직선 관통 감마선 버스트
- **밀리초 펄서** (Millisecond Pulsar) — 공속×3, 데미지×0.4

## Lv2 → Lv3 (최종 진화, 일부만)

| Lv2 출발 | Lv3 도달 | 효과 |
|----------|----------|------|
| 백색왜성 | Ia형 초신성 | 시한 폭발 (찬드라세카르 한계 초과) |
| 청색 초거성 | 블랙홀 | 근접 즉사 |
| 적색 초거성 | 초신성 잔해 | DoT 영역 |
| 맥동 변광성 | 행성상 성운 | 방어 감쇠 오라 |
| 밀리초 펄서 | 펄서 | 넉백 + 기절 |
| SGR 반복체 | 블랙홀 | 근접 즉사 |
