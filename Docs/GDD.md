> 천체 물리 기반 타워 디펜스  
> 플랫폼: 웹 (데스크톱/모바일 브라우저)  
> 스택: Babylon.js + TypeScript + Vite + Zustand  
> v2: 캠페인 구조, MVP 스코프 확정  
> 최종 수정: 2026-06-17

---

## 1. 컨셉

실제 천문 데이터(항성 카탈로그, 메시에 천체)를 기반으로 타워 스탯과 비주얼을 생성하는 타워 디펜스.
플레이어는 항성을 배치해 적(우주 위협체)을 저지하고, 성운으로 전장을 제어하며, 항성을 진화시켜 전략적 깊이를 만든다.

### 핵심 차별점

- 타워 스탯이 실제 천문 물리량(절대등급, 색지수, 광도계급)에서 파생
- ciToColor() 등 2Test1 셰이더 자산 재활용 → 아트 리소스 없이 절차적 비주얼
- HR도 기반 진화 트리로 타워 강화

## 5. 타워 시스템

### 5.1 스탯 산출 공식

타워의 기본 스탯은 실제 항성 물리량에서 파생한다.

```
화력(damage)   = f(절대등급)    — 밝을수록(작을수록) 강함
공격속도(rate) = f(분광형)      — O/B형 고온성: 빠른 연사, K/M형: 느린 한방
사거리(range)  = f(광도계급)    — 초거성(I): 장거리, 왜성(V): 단거리
색상(color)    = ciToColor(CI) — 색지수에서 자동 매핑
```

### 5.2 기본 타워 라인업

| 타워 | 분광형 | 역할 | CI 색상 | 비용(ISM) |
|------|--------|------|---------|-----------|
| 태양 (Sol) | G2V | 밸런스 입문. 중거리, 중화력 | 황백 | 낮음 |
| 프록시마 (Proxima) | M5V | 저비용 속사 + 랜덤 플레어 범위폭발 | 적색 | 최저 |
| 시리우스 (Sirius) | A1V | 중거리 메인 딜러 | 청백 | 중간 |
| 리겔 (Rigel) | B8Ia | 장거리 스나이퍼. 느리지만 고데미지 | 청색 | 높음 |
| 베텔게우스 (Betelgeuse) | M1Ia | 시한폭탄. 5웨이브 후 초신성 폭발 → 소멸 + 성운 잔해 | 적색 | 높음 |
| 마그네타 (Magnetar) | — | 관통 빔. 일직선 다수 적 관통 | 보라 | 높음 |

### 5.3 타워 셰이더 연출

2Test1의 CoreStarShader(NdotV limb darkening) 기반으로 타워별 변형:

- **기본**: 코어 글로우 구체 + ciToColor 자동 착색
- **프록시마**: 작은 코어 + 간헐적 플레어 파티클 버스트
- **리겔**: 넓은 가우시안 할로 + 느린 펄스 애니메이션
- **베텔게우스**: 맥동하는 코어 (radius oscillation), 폭발 시 화면 와이프 셰이더
- **마그네타**: 회전하는 빔 라인 셰이더 (각도 sweep)

### 5.4 HR도 진화 트리

진화 자원: 성간 물질 (ISM). 주계열성에서 시작, 2단계 분기.

```
주계열성 (Lv1) ─────────────────────────────────────────
    │
    ├── 적색거성 (Lv2)
    │     범위↑, 공속↓
    │     ├── 초신성 잔해 (Lv3)
    │     │     폭발 소멸 → 해당 위치에 성운 장판 영구 생성
    │     │     (초신성 잔해 성운: 지나가는 적에게 지속 데미지)
    │     │
    │     └── 행성상성운 (Lv3)
    │           타워 → 범위 디버프 오라로 변환 (적 방어력 감소)
    │           소멸하지 않고 영구 장판으로 전환
    │
    └── 청색거성 (Lv2)
          데미지↑, 비용↑
          ├── 블랙홀 (Lv3)
          │     좁은 사건의 지평선 내 적 즉사
          │     사거리 극히 짧지만 관통 불가 적 없음
          │
          └── 중성자별/펄서 (Lv3)
                360° 중력 파동 방사 (넉백/스턴)
                일정 리듬으로 CC, 라인 홀딩 특화
```

## 6. 성운 시스템 (유틸리티 타워)

MessierCatalog의 type 분류를 그대로 활용. 독립 공격 불가, 범위 효과 장판.

|------|--------|------|

| 퀘이사 제트 | 100% | 맵 가로 관통 직선 와이프. 경로 상 모든 적에게 최대 체력 비례 데미지 |

## 9. 맵 시스템

### 9.1 기본 구조

- 그리드 기반 타일맵 (탑다운 뷰, 고정 카메라)
- 적 이동 경로: 사전 정의 웨이포인트
- 타워 배치: 경로 외 타일에 1타워/1타일
- 성운 배치: 경로 위 타일에도 배치 가능 (장판 효과)

### 9.2 맵 환경 변수

맵마다 천체 환경이 다르고, 이것이 게임플레이에 영향:

| 환경 | 효과 |
|------|------|
| 성간 먼지 밀집 지역 | 모든 타워 사거리 -15%, 암흑성운 효과 +20% |
| 고에너지 영역 | 청색계(O/B) 타워 공격력 +20% |
| 중력 이상 지역 | 적 경로에 가속/감속 구간 존재 |

## 10. 캠페인: "소광(Extinction) — 별이 꺼지는 밤"

### 10.1 세계관

은하 외곽에서 밀려오는 **공허 조류(Void Tide)** — 지나간 자리의 별빛을 삼키는 정체불명의 파동.
플레이어는 **항성 건축가(Stellar Architect)** 로서, 남은 별들을 무기로 전환해 공허의 진군을 막는다.

캠페인은 태양계(가장 가까운 곳)에서 시작해 은하 중심부로 향하며, 공허 조류의 근원을 찾아 소멸시키는 구조.

### 10.4 맵 1-1: 태양계 외곽 (MVP 대상)

#### 배경 서사

공허 조류의 첫 징후가 오르트 구름에서 감지된다.
태양과 가장 가까운 이웃별 프록시마를 동원해 방어선을 구축하라.

#### 맵 레이아웃

```
  [스폰]                              [기지]
    ●━━━━━━┓                     ┏━━━━━★
           ┃                     ┃
           ┗━━━━━━━━━━━━━━━━━━━━━┛
     (단일 경로, S자 1회 굴곡)

  ○ = 타워 배치 가능 타일 (경로 양옆)
```

#### 사용 가능 타워

Sol, Proxima (2종만)

#### 웨이브 구성 (6웨이브)

| 웨이브 | 적 구성 | 의도 |
|--------|---------|------|
| 1 | 소행성 x5 | Sol 배치 학습 |
| 2 | 소행성 x8 | Sol 추가 배치 or 위치 조정 학습 |
| 3 | 소행성 x6 + 혜성 x2 | 빠른 적 등장. Proxima 해금 안내 |
| 4 | 혜성 x6 | Proxima 속사의 가치 체감 |
| 5 | 소행성 x10 + 혜성 x4 | 복합 웨이브. 배치 전략 시험 |
| 6 | 떠돌이 행성 x1 + 소행성 x8 | 미니 탱커 첫 등장. 집중 포화 학습 |

#### 클리어 보상

시리우스 항로 개방 (1-2 해금)

#### 튜토리얼 시퀀스

1. 웨이브 1 전: "성간 물질로 항성을 배치하세요" → Sol 강제 배치 가이드
2. 웨이브 1 완료: "항성이 자동으로 적을 공격합니다"
3. 웨이브 3 전: Proxima 해금 알림. "근접 고속 사격에 유리합니다"
4. 웨이브 5 전: "빌드 페이즈에서 타워를 추가하거나 위치를 조정하세요"

## 11. 웨이브 설계 가이드라인

범용 가이드라인 (캠페인 후반 및 커스텀 맵 기준):

- 웨이브 1~3: 소행성만. 태양 1~2개로 클리어 가능 → 튜토리얼 구간
- 웨이브 4~6: 혜성 혼합. 속사 타워(프록시마) 필요성 학습
- 웨이브 7~9: 떠돌이 행성 등장. 고화력(리겔) 또는 관통(마그네타) 필요
- 웨이브 10: 첫 보스 (GRB). 타워 비활 기믹 → 분산 배치 전략 학습
- 웨이브 11+: 암흑물질 등장. 펄서/마그네타 감지 타워 필수
- 웨이브 15+: 복합 웨이브 + 반물질 폭풍 보스
